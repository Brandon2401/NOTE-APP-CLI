# CLI Notes App

A simple command-line interface (CLI) application to manage your notes. You can create, view, search, edit, and delete notes quickly using a terminal.

 # Features

Add a Note: Save a note with a title and content.

View Notes: See all your saved notes.

Search Notes: Find notes by keywords in the title or content.

Edit Notes: Update the title or content of an existing note.

Delete Notes: Remove notes permanently.

# Installation

clone the repository 

navigate to the project folder

install dependancies

# Database

Uses SQLite for storing notes.

Database file: notes.db

Managed using SQLAlchemy ORM.
